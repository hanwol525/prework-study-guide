HTML

  • Hypertext Markup Language (HTML) gives content structure
  • HTML = "frame" for webpages/sites
  • "Head" element contains metadata/info to help render page correctly
  • "Body" element contains content shown to users; all elements we want users to see/interact with
  • "Section" elements are semantic elements that define a section of a doc that contains thematic content

CSS

  • CSS defines how webpage content is presented to users--colors, aesthetics, visual layout, organization, etc.
  • Inline CSS - applies to specific HTML tags; only changes particular page elements
  • Internal CSS style sheets - contain rules for webpage in head section HTML file
  • External CSS style sheets - separate files containing CSS rules; linked with "link" element in the "head" element of HTML (usually the preferred method)
  • Margin - space around outside of element
  • Padding - space inside element
  • Class attributes allow us to style certain elements with a class selector (e.g. "card")

Git

  • git status - checks current branch
  • git checkout -b branch-name - creates a new branch and switches to it
  • Git flow: git status (double check branch)->git add -A (stage changes)->git commit -m "messagehere" (commit to repository)->git pull origin main (make sure branch is up to date)->git push origin branch-name-here (push up all changes)->do The Thing in GitHub (open PR and merge)->git checkout main (get back to main branch)->git pull origin main (pull remote main branch back to local main branch)

JavaScript

  • JavaScript is used to facilitate user-website interaction
  • Variables - named containers that store data in code ("values")
  • Operators - mathematical symbol; produces results based on 2 values or variables
  • Conditionals - structures that test if an expression returns true or not
  • Functions - a body of code that executes when you call the function's name in your code
  • Event handlers - code structures that listen for activity in the browser and run code in response
  • Control flow - order in which a computer executes code in a script
  • Array - group of related data held in a single variable
  • Index - location numbers of data items in an array
